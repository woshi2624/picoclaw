# 📈 期货交易标准化规则模板

## 一、 给出的交易计划必须包含
*在给出交易指令之前，必须能清晰回答以下三个问题。*

| 维度 | 检查项 | 标准化填写要求 |
| :--- | :--- | :--- |
| **1. 交易类型** | 趋势 / 逆势 / 突破 | 明确是跟随趋势、摸顶/抄底还是追突破。 |
| **2. 核心逻辑** | 支撑压力 / 新闻消息 / 联动品种 | 理由必须包含：位置（在哪买）+ 逻辑（为什么买）。 |
| **3. 确认信号** | K线形态 / 关键指标 | 至少有三个信号共振（如：放量突破 + 站稳支撑）。 |
| **4. 入场价** | 执行价格 | 计划进入的具体数字，严禁“大概”、“差不多”。 |
| **5. 止损价** | **认赔离场位** | **最重要的一项！** 逻辑失效的价格点位。 |
| **6. 目标价** | 预期止盈位 | 确保盈亏比（预期收益/潜在亏损）大于 2:1。 |
| **7. 仓位管理** | 投入手数 | 新手期建议固定最小仓位（如：1手），严禁亏损加仓。 |
